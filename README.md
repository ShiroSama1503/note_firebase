# note_firebase

Flutter note app with Firebase environment configured for project `smart-note-p05032026`.

Firebase was configured with FlutterFire CLI:

```bash
flutterfire configure --project=smart-note-p05032026 --platforms=android,ios,web,windows --android-package-name=com.example.note_firebase --ios-bundle-id=com.example.noteFirebase --yes --overwrite-firebase-options
```
